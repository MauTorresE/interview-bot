# Phase 1 — UI Design Contract

> Visual and interaction contract for the Foundation phase: auth screens, dashboard shell, sidebar navigation, and empty-state pages. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn |
| Preset | base-nova (neutral base, CSS variables, RSC) |
| Component library | Radix UI (via shadcn) |
| Icon library | lucide-react |
| Font | Geist Sans (body/UI) + Geist Mono (code/data) |

**shadcn configuration** (replicate from prototype `components.json`):
- Style: `base-nova`
- Base color: `neutral`
- CSS variables: `true`
- RSC: `true`
- TSX: `true`
- Aliases: `@/components`, `@/components/ui`, `@/lib`, `@/hooks`

---

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-label gaps, inline padding |
| sm | 8px | Compact element spacing, input padding-x inner |
| md | 16px | Default element spacing, card padding, sidebar item padding |
| lg | 24px | Section padding within cards, form field vertical gaps |
| xl | 32px | Layout column gaps, sidebar width padding |
| 2xl | 48px | Major section breaks, page top/bottom padding |
| 3xl | 64px | Page-level vertical spacing, auth card centering margin |

Exceptions:
- Sidebar collapsed width: 56px (accommodates 40px icon button + 8px padding each side)
- Sidebar expanded width: 240px
- Touch targets: minimum 36px height for all interactive elements (desktop-first)

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 14px | 400 (regular) | 1.5 | Geist Sans |
| Label | 12px | 600 (semibold) | 1.4 | Geist Sans |
| Heading | 20px | 600 (semibold) | 1.2 | Geist Sans |
| Display | 28px | 600 (semibold) | 1.15 | Geist Sans |

Additional typographic rules:
- Nav items: 14px / 600 weight / 1.4 line-height
- Sidebar section labels (e.g., "Workspace"): 12px / 600 weight / uppercase / `text-muted-foreground` / letter-spacing 0.05em
- Monospace (code, IDs, technical values): Geist Mono at same size as surrounding text, 400 weight
- Maximum body text line length: 65ch (enforced via `max-w-prose` or explicit `max-width`)

## Color

Dark-first palette using shadcn CSS variable conventions. Light mode values deferred (not primary focus for v1 per D-07).

| Role | CSS Variable | Dark Value | Usage |
|------|-------------|------------|-------|
| Dominant (60%) | `--background` | `hsl(0 0% 3.9%)` | Page background (#0a0a0a) |
| Secondary (30%) | `--card` / `--sidebar` | `hsl(0 0% 7%)` | Cards, sidebar background, elevated surfaces (#121212) |
| Accent (10%) | `--primary` | `hsl(239 84% 67%)` | See reserved list below (#6366F1) |
| Destructive | `--destructive` | `hsl(0 84% 60%)` | Delete org, remove member, destructive confirmations (#ef4444) |

**Surface hierarchy (dark mode):**

| Level | Value | Usage |
|-------|-------|-------|
| Background | #0a0a0a | Page canvas |
| Surface 1 | #121212 | Sidebar, cards, dialogs |
| Surface 2 | #1a1a1a | Hover states on Surface 1 items, input backgrounds |
| Surface 3 | #262626 | Active/selected states, table row hover |
| Border | #262626 | Card borders, dividers, input borders |
| Border subtle | #1a1a1a | Separator lines within cards |

**Text hierarchy (dark mode):**

| Role | Value | Usage |
|------|-------|-------|
| Primary text | #fafafa | Headings, body copy, active nav items |
| Secondary text | #a1a1aa | Descriptions, placeholders, inactive nav items |
| Muted text | #71717a | Timestamps, metadata, helper text |
| Disabled text | #52525b | Disabled inputs, disabled buttons |

**Accent reserved for (explicit list):**
- Primary CTA buttons (filled background)
- Active sidebar nav item indicator (left border or background tint)
- Focus rings on interactive elements
- Links within body text
- Toggle/switch "on" state
- Progress indicators
- Badge highlights (e.g., "New", active status)

Accent is NOT used for: icons in nav (use secondary text), borders generally, section backgrounds, hover states on non-accent elements.

## Layout Contract

### Sidebar (D-01)

```
+---------------------------------------------------------+
| +--------+ +--------------------------------------+     |
| | Logo   | |                                      |     |
| |        | |         Content Area                 |     |
| +--------+ |                                      |     |
| | Campaigns| |  (adaptive width per D-02)          |     |
| | Reports| |                                      |     |
| | Settings| |                                      |     |
| |        | |                                      |     |
| |        | |                                      |     |
| +--------+ |                                      |     |
| |Org/User| |                                      |     |
| +--------+ +--------------------------------------+     |
+---------------------------------------------------------+
```

- Expanded: 240px wide, logo + text labels + org switcher
- Collapsed: 56px wide, icons only + tooltips
- Default state: expanded on desktop (>1024px), collapsed/sheet on mobile
- Collapse trigger: toggle button at bottom of sidebar, or automatically on viewport <1024px
- Animation: 200ms ease-out width transition
- Border: 1px right border using `--border` color

### Content Area (D-02)

- Full width: tables, dashboards, data-heavy views
- Max-width centered: forms, settings (max-width 640px, centered with auto margins)
- Page padding: 32px horizontal, 32px vertical on desktop; 16px on mobile

### Auth Pages

- Centered card layout, vertically and horizontally centered on viewport
- Card max-width: 400px
- Background: `--background` (#0a0a0a) -- no split-screen, no imagery
- Card: `--card` surface with `--border` border, 24px padding, 12px border-radius
- Logo centered above card, 48px margin-bottom to card

## Interaction Patterns

### Auth Flow States

| State | Visual Treatment |
|-------|-----------------|
| Form idle | Inputs with `--border` border, `--secondary-text` placeholder |
| Input focus | `--primary` (#6366F1) focus ring, 2px offset |
| Validation error | `--destructive` border, error message below field in 12px `--destructive` text |
| Submitting | Button shows spinner (lucide Loader2, animate-spin), disabled state |
| Success (signup) | Redirect to dashboard, sonner toast: "Cuenta creada" |
| Success (login) | Redirect to dashboard, no toast |
| Error | Error message above form in a subtle destructive banner |

### Sidebar Interactions

| State | Visual Treatment |
|-------|-----------------|
| Nav item default | `--secondary-text` icon + label, transparent background |
| Nav item hover | `--surface-3` background, `--primary-text` icon + label |
| Nav item active | `--primary` left border (3px), `--primary-text` icon + label, tinted background (primary at 10% opacity) |
| Collapse toggle | Chevron icon, rotates 180deg on toggle, 200ms ease |
| Org switcher | Dropdown menu with current org avatar + name, list of orgs, "Crear organizacion" option at bottom |

### Loading States

| View | Loading Pattern |
|------|----------------|
| Dashboard shell | Sidebar renders immediately (static), content area shows skeleton |
| Page content | Skeleton blocks matching expected content layout, 3 skeleton rows for lists |
| Auth form submit | Button spinner, inputs disabled, no skeleton |

### Empty States

| View | Pattern |
|------|---------|
| Campaigns page | Centered icon (lucide FolderOpen, 48px, muted), heading, body text, accent CTA button |
| Reports page | Centered icon (lucide FileBarChart, 48px, muted), heading, body text, no CTA (depends on campaigns) |
