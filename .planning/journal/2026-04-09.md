## Learnings & Insights

- **LiveKit Agents 1.5.2 event handlers**: `user_turn_completed` does NOT exist as a session event. Must override `Agent.on_user_turn_completed()` as a class method. `conversation_item_added` works on session but only fires for assistant messages reliably.
- **Event handlers must be registered BEFORE `session.start()`** — registering after silently loses events
- **LiveKit transcription**: Use `RoomEvent.TranscriptionReceived` (built-in), NOT data channels. Data channels are for custom messages (phase change, interview end).
- **Haiku 4.5 + prompt caching** is the right cost/quality point for voice AI — 3x cheaper than Sonnet with comparable question quality when prompt includes explicit probing techniques and few-shot examples
- **Prompt engineering for cheap LLMs**: Few-shot examples + explicit technique menus + strict constraints dramatically improves quality without retraining
- **Consultant role framing**: The agent should recopilate info, NOT propose solutions. Opportunities can still emerge naturally but the cierre just summarizes what was learned
- **VAD tuning for Spanish**: `min_silence_duration=1.5s` works better than 1.0s for deep thinking pauses
- **MXN pricing for Mexican small businesses**: USD anchors way too high. Rangos $15K-$35K MXN per phase, bundle $40K-$70K MXN total is the right positioning
- **Licensing vs ownership**: "Acceso completo + licencia de uso permanente" protects NSS IP while giving client everything they care about
- **Skill discovery requires YAML frontmatter**: SKILL.md needs `name:` and `description:` in frontmatter to be registered by Claude Code

## Product Evolution

- **Started as**: Generic multi-tenant research interview platform
- **Evolved to**: NSS discovery call automation tool (feeds proposals) — the proposal generation is a HUGE differentiator
- **Bigger insight**: The full pipeline is `discovery call → transcript → analysis → proposal → quote`. This can be a vertically integrated SaaS for consulting agencies, not just "another interview tool". The `/generate-proposal` skill is a glimpse of this future.
- **Pricing model insight**: For NSS as the client, fixed-price tiers like the one-pager ($3K/$7K/$15K USD) need to be rethought for Mexican small business market — MXN ranges work much better
