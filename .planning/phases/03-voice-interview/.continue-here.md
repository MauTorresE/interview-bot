## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| session.say() from timer loop | Calling session.say() at 80/90/100% causes voice overlap with LLM | blocking | All timing tiers below 110% ONLY update system prompt via _update_instructions(). Only 110% canned goodbye uses session.say() |
| Event handlers after session.start() | LiveKit 1.5.2 silently loses events registered after start | blocking | Register ALL @session.on() handlers BEFORE await session.start(agent=agent, room=ctx.room) |
| Fixed USD pricing for MX market | $5,000 USD (~$95K MXN) anchors way too high for Mexican small businesses | advisory | Use MXN ranges ($15K-$35K per phase) and always label as "estimados iniciales" |
| `session.room` access after aclose() | `_send_data` crashes with AttributeError if called after session closed | blocking | Check `hasattr(self.session, 'room') and self.session.room` before publishing data channel messages |
| Proposing during discovery call | Agent acting as consultant that proposes solutions during the call breaks NSS model | advisory | System prompt explicitly frames agent as info-gatherer only; opportunities captured post-call |
