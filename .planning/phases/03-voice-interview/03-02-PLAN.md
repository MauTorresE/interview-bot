The `EntrevistaAgent` class extends `Agent`:

1. **`__init__`** — receives instructions, interview_id, config, stores InterviewState
2. **`on_enter`** — called when agent session starts. Set up event handlers:
   - `session.on("user_turn_completed")` — save user transcript entry via `save_transcript_entry`, THEN send via data channel `self._send_data({"type": "transcript", "speaker": "client", "content": text, "elapsed_ms": ms})`
   - `session.on("conversation_item_added")` — save bot transcript entries (only when role is 'assistant'), THEN send via data channel `self._send_data({"type": "transcript", "speaker": "bot", "content": text, "elapsed_ms": ms})`
   - Room `data_received` handler for text_input and end_interview messages (from prototype lines 281-296)
3. **Function tools** (D-08):
   - `note_theme(theme, description, supporting_quote)` — saves insight, increments topics_count
   - `note_quote(quote, context, sentiment)` — saves insight
   - `note_sentiment(topic, sentiment, intensity)` — saves insight
   - `transition_phase(next_phase)` — transitions InterviewState, sends data channel message `{"type": "phase_change", "phase": next_phase}`, updates system prompt with new state_context
   - `end_interview(summary)` — marks state as ended, updates interview status in Supabase, sends `{"type": "interview_ended", "duration": ..., "topics_count": ...}` via data channel
4. **Time management** (D-15, D-16):
   - Before each agent turn, check InterviewState:
     - If `should_nudge`: inject prompt nudge "Comienza a cerrar el tema actual" into the conversation context, call `mark_nudged()`
     - If `should_force_close`: automatically call `transition_phase("closing")` and inject "Debes cerrar la entrevista ahora"
   - The `state_context` (elapsed time, current phase) is included in every turn via `_update_instructions()` which calls `build_system_prompt()` with the current state
5. **`_update_instructions()`** — rebuilds system prompt with current time_context from InterviewState and sets `self.instructions = new_prompt`
6. **`_send_data(data: dict)`** — sends JSON via data channel to all participants (from prototype pattern). Wrap in try/except.

The `entrypoint` function:
```python
async def entrypoint(ctx: JobContext):
    await ctx.connect(auto_subscribe=AutoSubscribe.AUDIO_ONLY)
    participant = await ctx.wait_for_participant()

    # Extract interview_id from room name (format: "interview-{uuid}")
    room_name = ctx.room.name
    interview_id = room_name.replace("interview-", "")

    # Load campaign config from Supabase (D-05)
    config = await load_interview_config(interview_id)

    # Build system prompt (D-06)
    state = InterviewState(config.duration_target_seconds)
    system_prompt = build_system_prompt(
        brief=config.research_brief,
        style=config.interviewer_style,
        duration=config.duration_target_minutes,
        state_context=state.time_context,
    )

    # Select TTS provider (D-07)
    if config.voice_persona.startswith("elevenlabs"):
        tts = elevenlabs.TTS(
            model="eleven_turbo_v2_5",
            voice=ELEVENLABS_VOICES.get(config.voice_persona, ELEVENLABS_VOICES["elevenlabs-sofia"]),
        )
    else:
        tts = VoxtralTTS(
            model="voxtral-mini-tts-2603",
            voice=VOXTRAL_VOICES.get(config.voice_persona, VOXTRAL_VOICES["voxtral-natalia"]),
        )

    agent = EntrevistaAgent(
        instructions=system_prompt,
        interview_id=interview_id,
        config=config,
        state=state,
    )

    session = AgentSession(
        stt=deepgram.STT(model="nova-3", language="es-419"),
        llm=anthropic.LLM(model="claude-sonnet-4-20250514"),
        tts=tts,
        vad=silero.VAD.load(
            min_speech_duration=0.3,
            min_silence_duration=1.0,  # D-34: longer endpointing for interviews
            prefix_padding_duration=0.3,  # D-34
        ),
        allow_interruptions=True,  # D-35
        false_interruption_timeout=2.0,  # D-35: prevents coughs/background noise from killing response
        resume_false_interruption=True,  # D-35: agent resumes speaking after false interruption
        user_away_timeout=12.0,  # D-37: silence re-engagement after 12s of no user speech
    )

    await session.start(ctx.room, participant)
```
